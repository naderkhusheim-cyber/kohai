# Third-party credits

## VOICEVOX engine

Kohai bundles the [VOICEVOX](https://voicevox.hiroshiba.jp/) speech synthesis engine — released under the **LGPL-3.0** license. Source: https://github.com/VOICEVOX/voicevox_engine

## Voice character

The default Kohai voice is **VOICEVOX:春日部つむぎ (VOICEVOX:Tsumugi Kasukabe)** — distributed under her own usage terms, which allow commercial use *with attribution*.

When using this voice, the app must display:

> **VOICEVOX:春日部つむぎ**

This attribution is shown in the in-app About / Credits panel.

For the full character usage terms see: https://tsumugi-official.studio.site/rule
